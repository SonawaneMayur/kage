# Analytics + Wrap-up

## Logs → Insights, with one SQL query each.

- **Pipeline SLA** — success rate, p90 duration, total volume
- **Medallion Health** — landing → gold survival %
- **Failure Root Cause** — error aggregation across jobs/tasks/datasets
- **Cost Attribution** — compute cost by pipeline × layer
- **Data Lineage Graph** — upstream/downstream dependency map
- **AI Resolution Suggestions** — opt-in via `ai_query()`
