# The Observability Gap

## Today's tools track **systems**, not **data**.

- Datadog / CloudWatch → CPU, memory, latency  ✅
- Pipeline lineage → who reads what? ❌
- Volume tracking → rows per layer? ❌
- Pipeline health → did it succeed, where did it fail? ❌
- Data freshness → is gold up to date? ❌

> "I can prove my cluster was healthy. I cannot prove my numbers were."
